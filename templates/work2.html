Password Checker Project

What is this project?

This was a complex, functional project which has a purpose to provide a secure way to check the security of a password. The script simply takes a system argument as a password, to then hash it (SHA1 hash), split the hashed password and send the one half of the password to an API to request and return the relevant data (how many times the password has been breached) The main feature of this project was to provide a super secure method of assessing security.

What did I learn?

This was a great project to test my overall Python knowledge. I further solidified my ability to write clean efficient code. This is a skill I will always be trying to improve. I was thoroughly testing my code to learn each stage and understand what was happening behind the scenes while learning how to handle / work around errors.
